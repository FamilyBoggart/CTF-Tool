# CTF-Tool
The ultimate ctf tool gathering all ctf codes and write-ups made


# Install
git clone git@github.com:FamilyBoggart/CTF-Tool.git
cd CTF-Tool
python -m venv ctf_venv
source ctf_venv/bin/activate
pip install -r requirements.txt
